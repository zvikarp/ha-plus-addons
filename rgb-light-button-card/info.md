## Features

- **Preset Color Buttons** - Quick access to 12 default colors (customizable)
- **Effect Controls** - Flash, Strobe, Fade, Smooth (customizable)
- **Brightness Slider** - Easy brightness adjustment
- **Power Toggle** - Simple ON/OFF button
- **Fully Configurable** - Customize colors, effects, and layout per light

## Quick Start

```yaml
type: custom:rgb-light-button-card
entity: light.bedroom_led_strip
```

## Default Colors

Based on typical cheap LED remote controls:
- Red, Green, Blue, White
- Orange, Cyan, Purple, Yellow  
- Pink, Lime, Magenta, Warm White

## Compatible With

Any Home Assistant light that supports RGB colors:
- AliExpress RGB strips
- Tuya lights
- ESPHome lights
- Zigbee RGB lights
- And more!
